PYRAMID PASS ONLINE — KHỞI ĐỘNG NHANH

CHƠI TRÊN INTERNET:
1. Giải nén thư mục.
2. Tạo một repository GitHub và upload toàn bộ file trong thư mục.
3. Vào Render > New > Blueprint.
4. Chọn repository vừa tạo. Render sẽ tự đọc render.yaml.
5. Sau khi deploy xong, mở link onrender.com.
6. Một người tạo phòng, gửi link mời cho ba người còn lại.
7. Đủ đúng 4 người thì chủ phòng bấm Bắt đầu trận đấu.

CHẠY THỬ TRÊN WINDOWS:
- Cài Node.js 20 trở lên.
- Nhấp đúp start_windows.bat.
- Hoặc mở Terminal trong thư mục và chạy:
    npm install
    npm start
- Sau đó mở http://localhost:3000

ĐIỀU KHIỂN TRÊN MỖI THIẾT BỊ:
- WASD hoặc phím mũi tên: di chuyển.
- Space: chuyền khi đang giữ Mặt Trời Vàng.
- Shift: lướt.
- Điện thoại có nút cảm ứng trên màn hình.
