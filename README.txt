PYRAMID PASS — STATIC SERVER FILES

Các file cần upload cùng một thư mục:
- index.html
- style.css
- game.js

Cách chạy:
1. Upload cả 3 file vào cùng một thư mục trên server.
2. Truy cập index.html qua HTTP/HTTPS.
3. Không đổi tên style.css hoặc game.js nếu chưa sửa đường dẫn trong index.html.

Ví dụ:
https://domain.com/pyramid-pass/index.html

Lưu ý:
- Đây là bản gameplay hiện tại chạy phía trình duyệt.
- Sprite nhân vật vẫn được nhúng trong game.js nên không cần thư mục assets riêng.
- Fullscreen cần thao tác click của người dùng, đúng theo quy định của trình duyệt.
